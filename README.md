# Chat-sederhana

### CaraMenjalankan
#### 
```
- Saya menggunakan Aplikasi Netbeans IDE 7.4
- setelah mendwonload seluruh file yang ada, jalankan pada aplikasi Netbeans IDE 7.4 (saya tidak mencoba pada versi berbeda)
- Jalankan dengan mengkombinasi tombol (SHIFT + F6), sebanyak 2 kali
- Setelah berjalan akan muncul box aplikasi 
- Pada Kedua box aplikasi yang mucul pergi ke salah satu box Aplikasi 
- Pergi ke TAB "server" pilih "Client"
- Setelah Ke dua box tersebut berbeda ganti dengan nama yang anda inginkan
- Setelah kedua nama sudah berbeda pergi ke box aplikasi dengan TAB "Server"
- Klik Tobol on dan berpindah pada box aplikasi kedua yang bernama "Client"
- Klik Tombol Hubungkan pada box aplikasa kedua 
- Akan keluar tulisan "Massukan Alamat IP server"
- Ketik tulisan (localhost) lalu ok
- Aplikasi sudah bisa di gunakan
